Impactos do git no mercado

<Dominância do git no mercado >

O git é inegavelmente um engolidor de mercado, sendo hoje, o sistema de versionamento de código mais utilizado no mundo.
Uma pesquisa do Stack overflow feita com desenvolvedores, mostrou que 88.4% de 74,298 participantes utilizavam o git.
O competidor mais próximo era o Subversion, com 16.6% de uso.

O git se tornou tão utilizado pelos desenvolvedores, porque foi pensado para ser uma ferramenta de fácil utilização,
além de ter diversas funções que outros sistemas de versiaonamento não possuem, facilitando assim o uso
tanto de desenvolvedores experientes quanto de desenvolvedores novatos.

<Algumas Vantagens do Git em relação a concorrência:>

  • Fácil de trabalhar
  • Repositórios menores
  • É possível verificar os status de suas modificações (git status)
