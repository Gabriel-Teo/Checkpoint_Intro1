O que é

  • • Git é o sistema de controle de versão open source mais usado no mundo atualmente.
  • • Usado para controlar o histórico de alterações de arquivos e principalmente de projetos de desenvolvimento de software.

O Que é um Sistema de Versionamento?

  • Ferramenta que ajuda equipes a gerenciar alterações em código fonte ao longo do tempo.
  • Mantém um registro de todas as versões do código, ou seja, de todas as modificações.

Como surgiu?

  • • Desenvolvido por Linus Torvalds, em 2005. Mesmo criador do Linux.
  • • Decidiu construir um sistema de controle de versão que tivesse a melhor performance.
  • • Sistema de controle de versão mais utilizado no mundo.
